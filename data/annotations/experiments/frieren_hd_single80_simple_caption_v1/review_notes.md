# Frieren L80 Simple Caption Dataset

- Dataset root: `data/datasets/frieren_hd_single80_simple_caption_v1`
- Total images: 80
- Metadata: `data/datasets/frieren_hd_single80_simple_caption_v1/metadata.jsonl`

Notes:
- Images are copied from the existing 80-image single-character dataset.
- Every sample uses the same simple caption: `sks_frieren, elf girl, long silver hair, pointy ears, white and gold robe`.
- This dataset is used only for the caption-organization comparison.
