# Experiment Configs

This folder contains fixed training configs for the final Frieren LoRA experiments.

Use these configs instead of editing `configs/train_sdxl_lora.yaml` directly.

| Config | Run name | Purpose |
| --- | --- | --- |
| `train_l80_structured_existing.yaml` | `frieren_sdxl_lora_hd80_single_full_v1` | Existing 80-image structured LoRA |
| `train_l100_structured.yaml` | `frieren_sdxl_lora_hd100_full_structured_v1` | 100-image structured LoRA candidate |
| `train_l80_simple_caption.yaml` | `frieren_sdxl_lora_hd80_single_simple_caption_v1` | 80-image simple-caption comparison |
